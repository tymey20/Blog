This is
CS50.

For my final project in CS50 I decided to create a digital resume/blog.
My goal is to use this as place to build my portfolio of visualizations and
code.

Some of the problem sets that utilized javascript and html inspired me to create my own blog. My plan is to use the blog to flesh out some of my ideas, showcase my growing portfolio of work, and serve as a spot to host my CV. Rather than create a site from scratch, I leveraged an open source template from HTML5up and then made some changes to the javascript and html in the template. The photos also came from an open source site - unsplash.com. The result is a fully responsive website that is as flexible as I want it to be.

I did some additional research and decided to host my web page on Github. This gave me the opportunity to learn version control with Git and Github through a Udacity course - How to Use Version Control in Git & GitHub. This has proven to be an extremely useful skill in constructing the website, and I'm sure will continue to be an asset as I advance my programming skills. I think that differentiating yourself in this space can be a great advantage, and I hope this site helps me to do just that.
